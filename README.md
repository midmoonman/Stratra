# STRATA // Creepy Analogue Audio Visualizer

An analogue-horror themed audio visualizer built specifically for videos, creepypastas, and simulated live AI interactions. It features a tactile 3D-ribbed disc soundwave and a WebGL-powered warping vortex background.

## 📺 Live Demo
If you have GitHub Pages or Vercel set up, you can access your visualizer directly online!

---

## 💀 Features

- **3D-Ribbed Disc Waveform**: Symmetrical overlapping vertical discs with 3D gradients, concentric grooves, and adjustable drop shadows.
- **WebGL Warp Vortex Background**: concentric ripples that distort and warp the background to the bass frequencies.
- **Multiple Inputs**:
  - *Default background*: Grey concentric tunnel (matches the concept image).
  - *Custom image*: Upload any background image to warp it.
  - *Webcam feed*: Warp your own camera stream in real-time.
- **Analogue Glitch Shader**: Chromatic aberration, dynamic horizontal line shifts, scanline grids, film grain, and heavy vignette.
- **Typewriter Dialogue Subtitles**: Sync words to audio peaks using offline onset peak detection for that authentic "live speech" typewriter feel.
- **Simulated HUD overlay**: flashing recording tags, uptime clocks, and a fluctuating *Hypnosis Index* that tracks bass intensity.

---

## ⌨️ Controls & Shortcuts

Press these keys while using the visualizer:
*   **`H`**: Toggle **Recording View** (instantly hides/shows the sidebar control panel for screen captures).
*   **`Space`**: Play / Pause the audio.

---

## 🛠 Setup & Run Locally

1. Clone this repository or download the files.
2. Open `index.html` in any web browser (Chrome, Edge, Firefox, or Safari).
3. Load an audio file, type or paste your transcript, adjust the sliders in the glassmorphic sidebar, and press **Connect (▶)**!
